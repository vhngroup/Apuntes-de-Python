# Apuntes-de-Python
Solo una lista de comandos y lines que me sacan de apuros
### Instalar una libreria para una version especiica de python
*  py -3.11 -m pip install "nombre de la libreria"
  
*  O si tenemos el whl descargado, usamos: `py -3.11 -m pip install "Tu/ruta/de/archivo.whl"`
*  O si es entorno virtual lo creamos de la siguiente manera: `py -3.11 -m pip install *nombre de la libreria*`
